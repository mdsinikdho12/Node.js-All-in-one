# Node.js

## 1 What is Node.js?

**Node.js** হলো একটি **JavaScript Runtime** যা ব্রাউজারের বাইরে সার্ভার-সাইডে JavaScript চালানোর জন্য ব্যবহার হয়।

- Event-driven, non-blocking I/O model
- Fast and lightweight
- Backend development, APIs, CLI tools ইত্যাদিতে ব্যবহার হয়

---

## 2️ How to Install Node.js

### Step 1: Download Node.js

- Visit [Node.js Official Website](https://nodejs.org/en/)
- Choose **LTS version** (recommended for most users)
- Download installer according to your OS (Windows, Mac, Linux)

---

### Step 2: Install Node.js (Windows)

1. ডাউনলোড করা `.msi` ফাইল রান করো
2. Next → Accept license → Next → Install
3. Finish

**Verify Installation:**

```bash
node -v
npm -v
```
